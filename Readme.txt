Приложение учёта рабочего времени и расчёта зарплаты сотрудников.

Используемые технологии и шаблоны:
-DevExpress
-Linq2db (http://linq2db.com/)
-Сохранение sql запросов в отдельные файлы (внедрённые ресурсы), собираемые во время выполнения в Dictionary.
Данный подход помимо возможности изоляции запросов в файлы и папки даёт возможность отладки запросов из Visual Studio
-FluentValidation (https://fluentvalidation.net/)
-Autofac (https://autofac.org/)
-MVP
-Паттерн Command
-DeepCloner
